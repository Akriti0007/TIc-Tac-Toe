Tic-Tac-Toe Game
A clean, interactive, and responsive implementation of the classic Tic-Tac-Toe game built using JavaScript, HTML, and CSS. This project focuses on DOM manipulation and game logic.

🚀 Features
Two-Player Gameplay: Players take turns as 'O' and 'X'.

Win Detection: The game automatically identifies winning patterns (rows, columns, and diagonals).

Draw Logic: Detects and announces a draw when all boxes are filled without a winner.

Interactive UI: Visual feedback for turns and game results.

Responsive Design: Uses CSS units to ensure the game board scales beautifully on both desktop and mobile screens.

Reset Functionality: Easy-to-use "Reset Game" and "New Game" buttons to restart the match at any time.

🛠️ Built With
HTML: For the structure of the game board and message containers.

CSS: For the styling, layout, and responsiveness.

JavaScript: For game logic, event listeners, and dynamic UI updates.

🎮 How to Play
Open the Game: Open index.html in any modern web browser.

Make a Move: Click on any empty box to place your mark ('O' goes first).

Win: Be the first to get three of your marks in a horizontal, vertical, or diagonal row.

Restart: Click the "Reset Game" or "New Game" button to clear the board.

📂 Project Structure
index.html: The HTML structure of the game.

style.css: The styling and responsive layout.

app.js: The JavaScript logic for handling turns, checking winners, and resetting the game.
